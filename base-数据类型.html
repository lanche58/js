问：数据类型有哪些?

答：

Number
String
Boolean
Undefined
Null
Object

问：检测变量的数据类型--typeof操作符?

答：

typeof  Number -- number
typeof  String -- string
typeof  Boolean -- boolean
typeof  Undefined -- undefined
typeof  Null -- object
typeof  Object -- object
typeof  Function -- function

注：typeof不能区分Null, Object

问：其他类型转化为布尔类型值为false的情况？

答：

Boolean(String) -- Boolean("") -> false
Boolean(Number) -- Boolean(0/NaN) -> false
Boolean(Undefined) -- Boolean(undefined) -> false
Boolean(Null) -- Boolean(null) -> false
